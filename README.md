# creative-writing

A workshop for novels that take unconventional forms — works composed by, around, or against language models.

Each subdirectory is a separate project with its own conventions, memories, and working notes. Agents spawned into a project should orient themselves by reading that project's `README.md` and `MEMORIES.md` before writing.

## Projects

- **`nabokov-2/`** — a Nabokov-style novel that unfolds as a sequence of chats between a user and a bot, with `MEMORIES.md` serving as the novel's living index and concordance.
