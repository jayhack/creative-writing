# MEMORIES

The living index of *nabokov-2*. Updated after every new chat. Read me before writing.

This file is itself part of the novel. A reader who never opens `chats/` should still be able to feel the shape of the book from these pages. A reader who reads only `chats/` should feel that something has been kept from them — and they would be right.
